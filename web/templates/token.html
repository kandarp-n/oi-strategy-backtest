Option A — Generate token via TOTP (recommended)

Requires TOTP to be enabled on your Dhan account.
Setup once: web.dhan.co → My Profile → Access DhanHQ APIs → enable TOTP and scan the QR with Google Authenticator/Authy/1Password.

Option B — Paste token manually

Get the token from web.dhan.co → My Profile → Access DhanHQ APIs → Generate Access Token, then paste here.

⚙️ How it works
  1. One-time setup: Enable TOTP on Dhan portal, scan the QR with an authenticator app.
  2. Daily: Open this page, enter Client ID, PIN, and the current 6-digit TOTP code from your authenticator.
  3. Click Generate. A new 24-hour token is fetched and saved to .env.
  4. The bot picks it up automatically — no restart needed.
🔐 Security
  • This UI runs on localhost:5005 only by default. Never expose to the internet.
  • Your PIN and TOTP are sent only to Dhan's auth server — not stored locally.
  • The access token IS stored in .env (so the bot can use it).
  • Tokens are valid for 24 hours and auto-expire.
